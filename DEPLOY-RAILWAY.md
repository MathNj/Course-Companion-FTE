# Quick Deploy to Railway (No Credit Card Required!)

Railway doesn't require payment info for their free tier. Let's deploy there instead.

## Step 1: Go to Railway

Visit: https://railway.app/new

## Step 2: Connect GitHub

1. Click "Deploy from GitHub repo"
2. Authorize Railway to access your GitHub
3. Select: `MathNj/Course-Companion-FTE`
4. Click "Deploy"

## Step 3: Configure Automatically

Railway will detect:
- ✅ Python/FastAPI app
- ✅ Requirements from pyproject.toml
- ✅ Port 8000 from environment

## Step 4: Add PostgreSQL Database

1. In Railway dashboard, click "New Service"
2. Select "Database"
3. Choose "PostgreSQL"
4. Railway sets up environment variables automatically

## Step 5: Set Environment Variables

Add these in Railway dashboard:

```
DATABASE_URL= (auto-generated by Railway)
JWT_SECRET_KEY=nFpDpFougSl6BkqqXwzsDmKHA6keETfSytpB8nPQlfw
APP_ENV=production
CORS_ORIGINS=https://chat.openai.com,https://chatgpt.com
LOG_LEVEL=INFO
```

## Step 6: Redeploy

Click "Deploy" again after adding database and env vars.

## Step 7: Run Migrations

Click on your app → "Migrations" tab → Run:
```
alembic upgrade head
```

## Your Production URL

Railway will provide a URL like:
```
https://course-companion-fte.up.railway.app
```
